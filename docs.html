How to create, download, and install your trading bots.

What is VANTIS AI?

VANTIS AI is a next-generation platform that lets you create AI-powered MetaTrader Expert Advisors (EAs) without any coding. Configure your strategy through an intuitive builder, describe your idea to the AI assistant, and we generate the MQL4 (.mq4) or MQL5 (.mq5) code for you. No subscriptions, no limits.

How to Create a Bot

  1. Go to the Bot Builder page.
  2. Give your bot a name and optional description.
  3. Choose a strategy: Grid, Martingale, Hedging, Trend Following, or Scalping.
  4. Configure money management (risk per trade, lot size, daily limits).
  5. Set trade management (stop loss, take profit, trailing stop).
  6. Define entry rules (buy/sell signals, trend filter, timeframe).
  7. Choose MT4 (.mq4) or MT5 (.mq5) format.
  8. Click Save & Download.

Supported Strategies

Grid Trading

Places buy and sell orders at predefined price intervals. Ideal for range-bound markets. Configure grid size, number of levels, and step distance.

Martingale

An aggressive recovery strategy that doubles the lot size after each losing trade. Configure the multiplier factor. Higher risk, higher potential recovery.

Hedging

Opens opposing positions to offset risk. Automatically places a hedge trade when a position moves against you. Useful for volatile markets.

Trend Following

Trades in the direction of the trend using Moving Average crossovers. Enters on trend confirmation and exits on reversal signals.

Scalping

Opens multiple small trades for quick profits using RSI-based entry signals. Designed for M5 timeframe with tight stop losses.

How to Install on MetaTrader 4 (MT4)

Follow these steps to install your generated .mq4 bot on MetaTrader 4:

1

Open MetaTrader 4

Launch MT4 on your computer. Make sure you are logged into your trading account.

2

Open the Data Folder

Click File > Open Data Folder. This opens the MT4 installation directory in your file explorer.

Tip: If you cannot find the File menu, press F4 to open MetaEditor, then click File > Open Data Folder from there.
3

Navigate to the Experts Folder

In the data folder, browse to: MQL4 > Experts (full path: %APPDATA%\MetaTrader 4\MQL4\Experts\)

4

Copy the .mq4 File

Copy the .mq4 file you downloaded from VANTIS AI into the Experts folder.

5

Refresh MetaTrader

In MT4, right-click in the Navigator panel (View > Navigator if hidden) and select Refresh. Your bot will appear under Expert Advisors.

Tip: If it still doesn't appear, restart MT4 completely.
6

Attach to a Chart

Drag the EA from Navigator > Expert Advisors onto any chart. The EA settings dialog will open — configure inputs as needed and click OK.

7

Enable Automated Trading

Make sure the AutoTrading button (top toolbar, usually with a play icon) is green/active. Also check that AutoTrading is enabled globally in Tools > Options > Expert Advisors > Allow Automated Trading.

How to Install on MetaTrader 5 (MT5)

Follow these steps to install your generated .mq5 bot on MetaTrader 5:

Open MetaTrader 5

Launch MT5 and log into your trading account.

Open the Data Folder

Click File > Open Data Folder to open the MT5 directory.

3

Navigate to the Experts Folder

Browse to: MQL5 > Experts (full path: %APPDATA%\MetaTrader 5\MQL5\Experts\)

4

Copy the .mq5 File

Copy the .mq5 file into the Experts folder.

5

Refresh MetaTrader

6

Attach and Activate

Drag onto a chart, configure inputs, and ensure the AutoTrading button is enabled.

Compiling Generated Code

The downloaded .mq4 / .mq5 files are source code. MetaTrader needs to compile them into .ex4 / .ex5 before they can run. This usually happens automatically when you attach the EA, but you can compile manually:

  1. After copying the file into the Experts folder, open MetaTrader.
  2. Press F4 to open MetaEditor.
  3. In MetaEditor, open the file from File > Open (navigate to the Experts folder).
  4. Click Compile (or press F7).
  5. The compiler output panel shows success or errors. A green checkmark means the .ex4/.ex5 was generated.
  6. If there are compilation errors, check the error message, adjust your bot settings, regenerate, and try again.

How to Enable AutoTrading

For your EA to execute trades automatically, AutoTrading must be enabled at multiple levels:

  1. Global setting: Tools > Options > Expert Advisors — check "Allow Automated Trading" and "Allow DLL imports" (if required by the EA).
  2. Chart level: After attaching the EA, click the AutoTrading button on the toolbar (it should turn green).
  3. EA level: In the EA settings dialog (drag onto chart), ensure the "Allow Automated Trading" checkbox is checked on the "Common" tab.
  4. Algo Trading button: In MT5, there is an "Algo Trading" button — ensure it is active (not crossed out).
Important: If any of these levels has AutoTrading disabled, your EA will be attached to the chart but will not place any trades. A smiley face icon on the top-right of the chart means the EA is active; a sad face means it is not.

Troubleshooting — Common Issues

EA not showing in Navigator

Refresh the Navigator (right-click > Refresh). If it still doesn't appear, restart MT4/MT5. Make sure the file is in the correct Experts folder with the right extension (.mq4 for MT4, .mq5 for MT5).

Sad face icon on chart (EA not running)

The EA is attached but cannot trade. Check: (1) AutoTrading is enabled in Tools > Options, (2) the AutoTrading button on the toolbar is green, (3) your account is a real or demo account (not "Investor" read-only mode).

Compilation errors

Open MetaEditor (F4), open the .mq4/.mq5 file, and compile (F7). The errors panel shows what's wrong. Common causes: missing curly braces, semicolons, or type mismatches. If errors persist, regenerate the bot with a simpler configuration.

"Expert Advisor cannot be tested" error

This usually means the EA uses features not supported in the Strategy Tester. Try testing the EA on a demo account instead by attaching it to a live chart.

"No history data" in Strategy Tester

Download historical data first: In MT4/MT5, open Tools > History Center, select your symbol and timeframe, click Download. Once complete, try the backtest again.

EA not opening trades

Verify: (1) Your trading account has sufficient margin, (2) the trade conditions are being met (check the Experts tab in the Terminal panel for log messages), (3) it's during the correct trading session for your symbol, (4) no trading holidays.

FAQ

Is this really free?

Yes. No subscriptions, no hidden fees. Create unlimited bots.

Can I edit a saved bot?

Yes. Go to My Bots and click Edit on any bot.

Where are my bots stored?

In your browser's localStorage. They're only accessible from this browser/device. We recommend downloading the code files for safekeeping.

Do I need coding skills?

No. The builder generates all the code for you.

Which MetaTrader versions are supported?

Both MT4 (MQL4) and MT5 (MQL5) are supported. Choose your platform before downloading.

Why does the EA have a sad face on the chart?

See the troubleshooting section above. Usually it means AutoTrading is disabled at some level (global, chart, or account).

Can I use the bot on multiple charts?

Yes. You can attach the same EA to multiple charts (different symbols or timeframes). Each chart runs an independent instance. Note that money management and position tracking may overlap — test thoroughly.
