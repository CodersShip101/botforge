How to create, download, and install your trading bots.

What is VANTIS AI?

VANTIS AI is a next-generation platform that lets you create AI-powered MetaTrader Expert Advisors (EAs) without any coding. Configure your strategy through an intuitive builder, describe your idea to the AI assistant, and we generate the MQL4 (.mq4) or MQL5 (.mq5) code for you. No subscriptions, no limits.

How to Create a Bot

  1. Go to the Bot Builder page.
  2. Give your bot a name (and optional description).
  3. Choose a strategy: Grid, Martingale, Hedging, Trend Following, or Scalping.
  4. Configure money management (risk per trade, lot size, daily limits).
  5. Set trade management (stop loss, take profit, trailing stop).
  6. Define entry rules (buy/sell signals, trend filter, timeframe).
  7. Choose MT4 (.mq4) or MT5 (.mq5) format.
  8. Click Save & Download.

Supported Strategies

Grid Trading

Places buy and sell orders at predefined price intervals. Ideal for range-bound markets. Configure grid size, number of levels, and step distance.

Martingale

An aggressive recovery strategy that doubles the lot size after each losing trade. Configure the multiplier factor. Higher risk, higher potential recovery.

Hedging

Opens opposing positions to offset risk. Automatically places a hedge trade when a position moves against you. Useful for volatile markets.

Trend Following

Trades in the direction of the trend using Moving Average crossovers. Enters on trend confirmation and exits on reversal signals.

Scalping

Opens multiple small trades for quick profits using RSI-based entry signals. Designed for M5 timeframe with tight stop losses.

How to Install on MetaTrader 4

  1. Open MetaTrader 4.
  2. Click File > Open Data Folder.
  3. Navigate to MQL4 > Experts.
  4. Copy the downloaded .mq4 file into the Experts folder.
  5. Restart MetaTrader 4 or right-click in Navigator and select Refresh.
  6. Drag your EA from Navigator > Expert Advisors onto a chart.
  7. Enable Automated Trading (AutoTrading button) and adjust input parameters as needed.

How to Install on MetaTrader 5

  1. Open MetaTrader 5.
  2. Click File > Open Data Folder.
  3. Navigate to MQL5 > Experts.
  4. Copy the downloaded .mq5 file into the Experts folder.
  5. Restart MetaTrader 5 or right-click in Navigator and select Refresh.
  6. Drag your EA from Navigator > Expert Advisors onto a chart.
  7. Enable Automated Trading and configure input parameters.

Compiling Generated Code

The downloaded .mq4 / .mq5 files are source code. MetaTrader needs to compile them into .ex4 / .ex5 before they can run. Here's how:

  1. After copying the file into the Experts folder, open MetaTrader.
  2. Press F4 to open MetaEditor.
  3. In MetaEditor, open the file from File > Open.
  4. Click Compile (or press F7).
  5. If successful, a compiled .ex4 / .ex5 file will appear in the Experts folder.
  6. If there are compilation errors, check the error log and adjust the inputs.
